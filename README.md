# BookRecommendationSoftware
Run BookRecommender.java

Enter the name and genre of your book. Then click "Add". Once the "Your Books" field is populated, click "Recommend". After this, you can optionally click "Write" to save to a file.

Viable genres are:
1. Fantasy
2. Action
3. Adventure
4. Comedy
5. Horror
6. Crafts/Hobbies
7. Young Adult Fiction
8.
